Semi-Restore

Testing







Since iOS 8.1.1 is already in beta, SemiRestore needs to be updated for iOS 8 ASAP. However, this means that SemiRestore will need to be tested on every device model available. For this, I need your help.

How can I help?


If you have a PC running Windows XP and higher, Ubuntu 14.04, Linux Mint 17, or Elementary OS Freya and higher and one of the devices that isn't marked as already tested, please download f0recast for Windows (or a Linux equivilant) and send a screenshot of it/terminal output from it with your device plugged in to coolstarorganization@gmail.com. Then, I'll notify you when a build is ready to test. All of this will happen before 8.1's signing windows are closed.
